Geographic Map-Feature Idendification Quiz v2.0
===============================================

Introduction
------------

(IN-PROGRESS)
This map quiz will enable educators to create quizzes around identifying features of maps.  There will be both a student portal and administrative portal.  That latter, from which, will allow educators to login and dynamically create the quizzes.

The front-end is being built using AngularJS v.1.5.-, while the API will likely be a combination of NodeJS and SQL -TBD.
